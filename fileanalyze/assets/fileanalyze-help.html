File Analyze - How To Tutorial

Purpose, features, tabs, and step-by-step usage guide.

Purpose

File Analyze helps you load a delimited dataset (CSV/pipe/tab), profile columns, visualize trends, inspect quality risks, and explore records in a powerful interactive Data Grid.

Key Features

  • Fast metadata-driven profiling with on-hover quick stats.
  • Chart generation with auto chart recommendations.
  • Quality Scorecard and Top Findings insight summary.
  • Advanced Data Grid with dimension/measure filters and column hide/unhide drawer.
  • Copy grid to clipboard and export filtered CSV.
  • Large-data safeguards and server-side grid mode for very large results.

Tabs Overview

  • Visual Analysis Tab: choose fields, build chart, inspect Field Info and Quality Scorecard.
  • Data Grid Tab: apply multi-row filters, sort/hide columns, review row-level records.

Step 1 - Load Data

  1. Enter full file path in Input File Path.
  2. Select delimiter: comma, pipe, or tab.
  3. Click Load + Profile.

Tip: metadata sidecar is expected (for example, yourfile.csv_Meta) to drive field types and labels.

Step 2 - Generate Analysis

  1. Go to Visual Analysis tab.
  2. Select Dimension, Measure, optional Color Dimension.
  3. Choose Chart Type (or let app auto-recommend).
  4. Click Generate Chart.

Below the chart you will see:

  • Field Info (selected field metrics/top values)
  • Quality Scorecard (null %, invalid parse %, duplicate PK %, warnings)
  • Top Findings summary insights

Step 3 - Filter Data in Data Grid

  1. Go to Data Grid tab.
  2. Dimension side (left): choose field + value set.
  3. Measure side (right): choose field + range set.
  4. Use Add More Dimension Filters / Add More Measure Filters for extra rules.
  5. Click Apply Filters to refresh grid.
  • Field selectors are cascading: a selected field is removed from other rows.
  • You can sort column names with Sort the Column Names.

Column Controls and Copy/Export

  • Click the hamburger icon (☰) to open column hide/unhide drawer.
  • Use drawer search + Select All / Deselect All.
  • Click clipboard icon (📋) to copy visible grid view to clipboard (Excel-friendly).
  • Click Export Filtered CSV to download full filtered result.

Large File Behavior

  • For very large filtered outputs, the app activates a safety guard and shows a warning banner.
  • Grid may switch to server-side mode so only paged blocks are loaded in browser.
  • Use CSV export for complete full-data output when preview is sampled.
